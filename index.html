Aplicaçõesem WebSig no Ordenamento do Território, 2020/21– 1ª edição

MSIGOT, U.Porto

Objetivos

Esta disciplina tem como objetivos dotar os estudantes de:

Descrever os princípios de funcionamento da Interet, da World Wide Web (WWW) e das redes de comunicação.

  • Conceber, estruturar e produzir conteúdos para a Web utilizando as linguagens de desenvolvimento Web ( HTML, CSS e Javascript ) em problemáticas específicas.
  • Desenvolver competências para a disponibilização e visualização de informação geográfica na Web, utilizando framework leaflet.
  • Elaborar um protótipo de um WEBSIG aplicado a um problema de Ordenamento do Território.
